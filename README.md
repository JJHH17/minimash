# minimash
A hub for mini games, including Rock, Paper, Scissors, Etch-a-sketch, and TicTacToe

# Project Overview:
- Project cotains 3 minigames: TicTacToe, Rock, Paper, Scissors and a multiplayer etch-a-sketch game.
- Project is created using vanilla Javascript, HTML and CSS.
- Bundled using Webpack

# Installation Steps (for Development) - Node packages:
- Webpack Installation:
```npm install --save-dev webpack webpack-cli```

- HTML Handling:
```npm install --save-dev html-webpack-plugin```

- CSS Handling:
```npm install --save-dev style-loader css-loader```

- Image Handling:
```npm install --save-dev html-loader```

- Local Host (8080) Server (Optional):
```npm install --save-dev webpack-dev-server```
